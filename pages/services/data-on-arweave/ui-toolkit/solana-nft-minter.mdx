# Solana NFT minter

The [Solana NFT Minter component](https://provenance-toolkit.irys.xyz/solana-nft-minter):

1. Uploads an image to Irys
2. Mints the image as a Solana NFT using the Helius SDK

<img
	className="border border-[#FEF4EE] rounded-xl mt-5 md:w-3/6 w-full"
	src="/img/provenance-toolkit/solana-minter.png"
/>

## Minting

This component uses the [Helius SDK](https://www.helius.dev/) to mint the uploaded image as a compressed NFT on Solana.

To use this component, you must create a [Helius developer account](https://dev.helius.xyz/dashboard/app) first.

<img className="border border-[#FEF4EE] rounded-xl mt-5 w-full" src="/img/provenance-toolkit/heluis-backend.png" />

Then, add your Helius API key as the value for the `NEXT_PUBLIC_HELIUS_API` field in the Provenance Toolkit's `.env.local` file.
